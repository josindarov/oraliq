LeNet, AlexNet, VGG Arxitekturalari

LeNet, AlexNet va VGG konvolyutsion neyron tarmoqlarning (CNN) mashhur arxitekturalaridir. Har biri ma’lum vazifalarni hal qilish uchun mo‘ljallangan va ular tasvirlarni tanish, qayta ishlash va tasniflashda keng qo‘llaniladi.

1. LeNet

  • Yaratilgan yil: 1989
  • Muallifi: Yann LeCun
  • Vazifasi: Raqamlarni tanish.
  • Qo‘llanilish sohasi: Qo‘l yozuvi va bank cheklarini raqamli tahlil qilish.

LeNet oddiy arxitektura bo‘lib, unda ketma-ket konvolyutsiya, pooling va to‘liq bog‘langan qatlamlar mavjud. Ushbu arxitektura MNIST datasetida 28x28 o‘lchamdagi tasvirlarni tasniflashda samarali ishlaydi.

Arxitektura Tuzilmasi:

Input → Conv → Pool → Conv → Pool → FC → Output

2. AlexNet

  • Yaratilgan yil: 2012
  • Muallifi: Alex Krizhevsky, Ilya Sutskever, Geoffrey Hinton
  • Vazifasi: Katta hajmdagi tasvirlarni tasniflash.
  • Qo‘llanilish sohasi: ImageNet musobaqalari va umumiy tasvirlarni tanish tizimlari.

AlexNet LeNetdan farqli o‘laroq, chuqurroq qatlamlarga va ReLU aktivatsiya funktsiyalariga ega bo‘lib, GPU yordamida parallel hisoblash imkonini beradi.

Arxitektura Tuzilmasi:

Input → Conv → ReLU → Pool → Conv → ReLU → Pool → FC → Output

Innovatsiyalar:

  • Dropout qatlami bilan overfittingni kamaytirish.
  • ReLU aktivatsiyasi bilan hisoblashni tezlashtirish.

3. VGG

  • Yaratilgan yil: 2014
  • Muallifi: Visual Geometry Group (VGG), Oksford universiteti.
  • Vazifasi: Yuqori aniqlikdagi tasvirlarni tasniflash.
  • Qo‘llanilish sohasi: Katta miqyosli tasniflash va xususiyatlarni chiqarish.

VGG chuqur konvolyutsion tarmoqlarni qo‘llagan holda har bir qatlamda 3x3 o‘lchamdagi filtrlar ishlatadi. Bu arxitektura oddiy, lekin juda kuchli dizaynga ega.

Arxitektura Tuzilmasi:

Input → [Conv → ReLU] x N → Pool → [Conv → ReLU] x M → Pool → FC → Output

Innovatsiyalar:

  • Chuqurlikni oshirish orqali yuqori aniqlik.
  • Kichikroq filtrlardan foydalanib, ko‘proq kontekstni qamrab olish.

Xulosa

LeNet, AlexNet va VGG o‘z davrining muhim arxitekturalari bo‘lib, tasvirni qayta ishlashning turli darajadagi vazifalarida muvaffaqiyatli qo‘llaniladi. Ular CNN arxitekturalari rivojlanishida asosiy bosqichlarni ifodalaydi.
